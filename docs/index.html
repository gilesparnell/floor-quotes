Concept Stage Floor Quotes — v0.1.3

Flooring quotes,
without the back-and-forth.

A quote-ready sales and operations assistant for independent flooring installers. Customers scan or describe their room, explore real products, and receive an indicative price. Installers get a structured lead ready to act on.

Technical Architecture

Three-path input system: iPhone LiDAR/RoomPlan scan, room photo upload, and 2D floor plan parsing. All paths normalise into a shared project record that feeds the quote engine, ordering workflow, and installer dashboard. AWS deployment view included.

  • LiDAR, photo, and floor-plan inputs
  • Shared project record model
  • AWS Fargate + SQS + RDS stack
  • MVP vs. full-scale path
