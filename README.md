# Floor Quotes

Floor Quotes is a concept site for a flooring quote automation product.

The product idea:

> A quote-ready spatial commerce platform for local flooring businesses: capture room evidence, visualise product options, estimate quantity and cost, and convert the customer into a structured quote or booking.

## Site

The public webpage explains the motivation for the product and links to the current written and visual artifacts.

```bash
npm install
npm run verify
npm run build
npm run dev
```

## Current conclusion

The generic flooring visualizer space is already crowded.

However, there may be a narrower opportunity around:

- local trades and small flooring installers;
- quote-ready room scanning;
- product catalogue aggregation;
- accurate material quantity estimation;
- installer workflow, lead qualification, and quoting;
- lower-cost alternative to enterprise visualizer platforms;
- a portfolio-grade spatial AI project that shows computer vision, 3D, ecommerce, and business process design.
